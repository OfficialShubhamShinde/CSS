What is CSS Selector:

  • CSS selector are used to find the element whose property will be set
  • Selectors are used to target the html element
  • Selectors makes it easy for us to easily target single/multiple HTML element in the markup

Types of Basic CSS Selector:

  • CSS Element Selector
  • CSS id selector
  • CSS Class Selector
  • The CSS Grouping Selector
