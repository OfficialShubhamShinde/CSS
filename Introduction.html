What is CSS:

  • CSS gives style to raw HTML
  • CSS Stands for Cascading Style Sheets
  • CSS is used to give style to our page
  • CSS is used to make websites responsive
  • HTML is used to stucture website
  • CSS takes responsibility of design
  • Comment in CSS: /* */

Role of CSS

  • Cascading Style Sheets which is known as CSS is a style sheet language that used to handle the presentation of the web page containing HTML
  • It makes our website Beautiful and modern looking

Syntax of CSS

p{color : red}
p for target(selector)
color for which property u want to add(property)


Three ways to add CSS in markup(HTML Code)

  • Inline CSS: CSS is added to the elements directly using style attribute
  • Internal CSS: CSS is kept inside the head tags in style tag
  • External CSS: CSS is kept separately inside a .CSS style sheet
